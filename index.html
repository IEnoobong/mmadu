Mmadu

Introduction

Mmadu is a set of tools and services that helps developers manage their application users.

Getting Started

Mmadu can be easily started with docker compose.

Steps to Run Mmadu User Service

To run Mmadu User service, take the following steps:

  1. Create a folder (we will use mmadu as the created folder)

    mkdir mmadu && cd mmadu

  2. Download the docker compose file

    wget https://raw.githubusercontent.com/geraldoyudo/mmadu/master/docker/docker-compose.yml

  3. Download the default environment file

    wget https://raw.githubusercontent.com/geraldoyudo/mmadu/master/docker/.env

  4. Make the necessary modifications to suite your configuration (you can use it as is)

  5. Start the services

    docker-compose up
